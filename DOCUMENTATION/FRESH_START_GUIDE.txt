================================================================================
                    COMMIT MESSAGE FORMAT
================================================================================

Use conventional commit format:

feat: Add new feature
  Capitalize first letter
  Be clear and concise
  Example: "feat: add two-factor authentication"

fix: Fix a bug
  Example: "fix: resolve memory leak in screen capture"

refactor: Improve code without changing functionality
  Example: "refactor: simplify database connection logic"

docs: Update documentation
  Example: "docs: update API endpoint documentation"

test: Add or improve tests
  Example: "test: add unit tests for payment processing"

chore: Maintenance tasks
  Example: "chore: update dependencies to latest versions"

style: Code formatting changes
  Example: "style: add missing type annotations"

perf: Performance improvements
  Example: "perf: optimize image processing algorithm"

================================================================================
                    USEFUL GIT COMMANDS
================================================================================

See what you changed:
  git diff                          # See unstaged changes
  git diff --staged                 # See staged changes
  git log --oneline -10             # See recent commits

See branch status:
  git status                        # Current branch status
  git branch                        # List local branches
  git branch -a                     # List all branches (local + remote)
  git log --oneline --graph -10     # Visual commit history

Work with remotes:
  git fetch origin                  # Get latest from GitHub
  git pull origin branch-name       # Fetch and merge
  git push origin branch-name       # Push to GitHub

Undo changes:
  git restore file.py               # Discard changes to file
  git restore --staged file.py      # Unstage file
  git reset HEAD~1                  # Undo last commit (keep changes)
  git reset --hard HEAD~1           # Undo last commit (discard changes)

Create from existing branch:
  git checkout -b new-branch existing-branch

Rename branch:
  git branch -m old-name new-name

Delete branch:
  git branch -d branch-name         # Local
  git push origin --delete branch-name  # Remote

Stash work temporarily:
  git stash                         # Save changes temporarily
  git stash list                    # See stashed changes
  git stash pop                     # Restore stashed changes
